## 🔧 Easy to Customize

You can easily modify animations:

### Change Animation Speed
```css
/* Make float animation slower */
.hero-image-animate {
    animation: heroImageFloat 6s infinite;  /* was 4s */
}
```

### Change Colors
```css
/* Edit the color in @keyframes */
@keyframes textGlow {
    50% {
        text-shadow: 0 0 20px rgba(255, 100, 0, 0.8);
    }
}
```

### Disable Specific Animation
```html
<div style="animation: none;">No animation here</div>
```

### Make Everything Slower
```css
* {
    animation-duration: 2s !important;
}
```
