## How to Use

### To View Animations
1. Open browser
2. Navigate to your homepage
3. Watch animations on page load
4. Hover on cards and buttons to see interactions

### To Customize
1. Edit `/assets/main.css` for animation timing
2. Edit `/index.html` to apply/remove animations
3. Refresh browser (Ctrl+Shift+R) to see changes

### To Modify Speed
```css
/* Find the animation in main.css */
.hero-image-animate {
    animation: heroImageFloat 4s infinite;
}

/* Change 4s to desired duration */
.hero-image-animate {
    animation: heroImageFloat 6s infinite;  /* Now slower */
}
```

### To Change Colors
```css
/* Find the @keyframes in main.css */
@keyframes textGlow {
    50% {
        text-shadow: 0 0 20px rgba(6, 182, 212, 0.8);  /* Change color here */
    }
}
```

## Support & Troubleshooting

### Common Issues

**Q: Animations not showing?**
A: Clear cache (Ctrl+Shift+Delete) and hard refresh (Ctrl+Shift+R)

**Q: Animations too fast?**
A: Change duration value in main.css (e.g., `4s` → `6s`)

**Q: Want to disable animations?**
A: Add `animation: none !important;` to element style

**Q: Need different colors?**
A: Edit RGB values in `@keyframes` definitions

**Q: Mobile animations stuttering?**
A: Check device performance, try closing other apps
